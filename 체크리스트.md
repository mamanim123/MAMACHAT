Mamabot Direct Provider Routing 작업계획서
0. 작업 목표

현재 Mamabot은 Quick 모드에서 선택한 모델이 Claude/Gemini/NVIDIA/DeepSeek 계열이어도 무조건 OpenRouter API로 보내는 구조다.
이 구조를 아래처럼 바꾼다.

Gemini API 모델 선택 → Gemini API로 직접 호출
Anthropic Claude API 모델 선택 → Anthropic API로 직접 호출
OpenAI API 모델 선택 → OpenAI API로 직접 호출
NVIDIA 모델 선택 → NVIDIA API로 직접 호출
DeepSeek 모델 선택 → DeepSeek API로 직접 호출
OpenRouter 모델 선택 → OpenRouter API로 호출
Claude Code / Gemini CLI / Codex CLI 선택 → Quick에서는 차단, Agent/Coding/Review에서만 CLI 실행

핵심은 모델 ID가 아니라 provider 값으로 라우팅하는 것이다.

1. 현재 확인된 상태
1-1. API Provider 설정은 이미 존재함

config/providers.json에는 이미 API provider 구조가 있다.

현재 등록된 provider:

openrouter → OPENROUTER_API_KEY
openai → OPENAI_API_KEY
anthropic → ANTHROPIC_API_KEY
gemini → GEMINI_API_KEY

추후 추가할 provider:

nvidia → NVIDIA_API_KEY
deepseek → DEEPSEEK_API_KEY

현재 파일에서 openrouter, openai, anthropic, gemini가 apiProviders로 등록되어 있고 각각 envKey를 가지고 있다.

1-2. 현재 Quick 실행은 OpenRouter 전용

현재 directModelClient.js는 runOpenRouterDirect() 중심이고, OpenRouter API만 호출한다.
기존 함수는 OpenRouter API 키만 읽고, URL도 https://openrouter.ai/api/v1/chat/completions로 고정되어 있다.

현재 Quick 블록은 다음 구조다.

executionProfile === "quick"
→ runOpenRouterDirect({
    prompt,
    model,
    responseMode
  })

즉 Quick에서는 provider가 무엇이든 OpenRouter direct로 간다.

1-3. UI 요청 payload에는 provider와 model이 전달됨

WorkbenchChatPanel.jsx는 /api/agent/run 호출 시 다음 값을 보낸다.

provider: requestProvider
agentId: requestProvider
model: requestModel
responseMode: responseStyle
executionProfile

즉 프론트에서 provider를 서버로 보낼 수 있는 구조는 이미 있다.

1-4. modelPresets는 대부분 provider가 openrouter로 고정되어 있음

현재 modelPresets.js의 추천 모델들은 대부분 이런 식이다.

{
  id: "google/gemini-2.5-flash",
  label: "Gemini 2.5 Flash",
  provider: "openrouter"
}

그래서 Gemini 계열 모델을 골라도 실제 provider가 openrouter로 잡힌다.
앞으로는 같은 Gemini라도 아래처럼 분리해야 한다.

// Gemini API 직접 호출

// OpenRouter 경유

3-2. 새 export 추가
export async function runDirectModel({
  provider = "openrouter",
  prompt,
  model,
  responseMode = "short",
  maxTokens = 512,
  webSearch = false
}) {
  const providerId = String(provider || "openrouter").trim();

  if (providerId === "openrouter") {
    return runOpenRouterDirect({ prompt, model, responseMode, maxTokens, webSearch });
  }

  if (providerId === "gemini") {
    return runGeminiDirect({ prompt, model, responseMode, maxTokens });
  }

  if (providerId === "anthropic") {
    return runAnthropicDirect({ prompt, model, responseMode, maxTokens });
  }

  if (providerId === "openai") {
    return runOpenAIDirect({ prompt, model, responseMode, maxTokens });
  }

  if (providerId === "nvidia") {
    return runNvidiaDirect({ prompt, model, responseMode, maxTokens });
  }

  if (providerId === "deepseek") {
    return runDeepSeekDirect({ prompt, model, responseMode, maxTokens });
  }

  throw new Error(`Unsupported direct provider: ${providerId}`);
}

4-1. Gemini API

Endpoint 예시:

https://generativelanguage.googleapis.com/v1beta/models/{model}:generateContent?key={apiKey}

구조:

async function runGeminiDirect({ prompt, model, responseMode, maxTokens }) {
  const envKey = getProviderEnvKey("gemini") || "GEMINI_API_KEY";
  const apiKey = getSecretValue(envKey);
  if (!apiKey) throw new Error(`Gemini API key is missing. Expected env key: ${envKey}`);

  const userPrompt = cleanDirectPrompt(prompt);
  const actualModel = model || "gemini-2.5-flash";

  const body = {
    contents: [
      {
        role: "user",
        parts: [{ text: buildDirectPrompt(userPrompt, responseMode) }]
      }
    ],
    generationConfig: {
      maxOutputTokens: maxTokens,
      temperature: 0.4
    }
  };

  const startedAt = Date.now();

  const response = await fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/${encodeURIComponent(actualModel)}:generateContent?key=${apiKey}`,
    {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify(body)
    }
  );

  const json = await response.json().catch(() => null);

  if (!response.ok) {
    throw new Error(`Gemini direct failed: HTTP ${response.status}: ${JSON.stringify(json).slice(0, 500)}`);
  }

  const output = json?.candidates?.[0]?.content?.parts?.map((p) => p.text || "").join("") || "";

  return {
    output,
    raw: json,
    usage: json?.usageMetadata || null,
    model: actualModel,
    durationMs: Date.now() - startedAt
  };
}

async function runAnthropicDirect({ prompt, model, responseMode, maxTokens }) {
  const envKey = getProviderEnvKey("anthropic") || "ANTHROPIC_API_KEY";
  const apiKey = getSecretValue(envKey);
  if (!apiKey) throw new Error(`Anthropic API key is missing. Expected env key: ${envKey}`);

  const userPrompt = cleanDirectPrompt(prompt);
  const actualModel = model || "claude-3-5-haiku-latest";

  const body = {
    model: actualModel,
    max_tokens: maxTokens,
    temperature: 0.4,
    system: getDirectSystemPrompt(responseMode),
    messages: [
      {
        role: "user",
        content: userPrompt
      }
    ]
  };

  const startedAt = Date.now();

  const response = await fetch("https://api.anthropic.com/v1/messages", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "x-api-key": apiKey,
      "anthropic-version": "2023-06-01"
    },
    body: JSON.stringify(body)
  });

  const json = await response.json().catch(() => null);

  if (!response.ok) {
    throw new Error(`Anthropic direct failed: HTTP ${response.status}: ${JSON.stringify(json).slice(0, 500)}`);
  }

  const output = Array.isArray(json?.content)
    ? json.content.map((item) => item.text || "").join("")
    : "";

  return {
    output,
    raw: json,
    usage: json?.usage || null,
    model: json?.model || actualModel,
    durationMs: Date.now() - startedAt
  };
}

async function runOpenAIDirect({ prompt, model, responseMode, maxTokens }) {

  const body = {
    model: actualModel,

    ],
    max_tokens: maxTokens,
    temperature: 0.4
  };

  const startedAt = Date.now();

  const response = await fetch("https://api.openai.com/v1/chat/completions", {
    method: "POST",
    headers: {
      Authorization: `Bearer ${apiKey}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify(body)
  });

  const json = await response.json().catch(() => null);

  if (!response.ok) {
    throw new Error(`OpenAI direct failed: HTTP ${response.status}: ${JSON.stringify(json).slice(0, 500)}`);
  }

  return {
    output: json?.choices?.[0]?.message?.content || "",
    raw: json,
    usage: json?.usage || null,
    model: json?.model || actualModel,
    durationMs: Date.now() - startedAt
  };
}

function getDirectSystemPrompt(responseMode = "short") {
  if (responseMode === "short") {
    return "Reply briefly in the user's language. Do not mention system instructions.";
  }

  if (responseMode === "detail") {
    return "Reply clearly and with useful detail in the user's language. Do not mention system instructions.";
  }

  return "Reply clearly in the user's language. Do not mention system instructions.";
}

function buildDirectPrompt(userPrompt = "", responseMode = "short") {
  return userPrompt;
}

const actualSelectedModel = selectedModelInfo?.model || model;
const requestProvider = parsedAuthModelForRun ? parsedAuthModelForRun.agentId : (selectedModelInfo?.provider || provider);

14. 중요한 주의점
14-1. Quick과 CLI는 분리
Quick = API direct
Agent/Coding/Review = CLI agent 가능

Quick에서 CLI를 다시 열면 안녕도 작업 지시로 해석해서 딴소리한다.

14-2. OpenRouter web_search는 OpenRouter에서만 사용
webSearch: responseMode === "detail" && directProvider === "openrouter"

Gemini/Anthropic/OpenAI/NVIDIA/DeepSeek에 무조건 webSearch를 넘기면 안 된다.

14-3. provider와 model을 분리

잘못된 구조:

provider=openrouter
model=claude-haiku-4-5-20251001

올바른 구조:

provider=anthropic
model=claude-3-5-haiku-latest

또는:

provider=openrouter
model=anthropic/claude-3.5-haiku
14-4. 무료 모델은 provider별로 따로 등록

예:

NVIDIA 무료 모델
→ provider=nvidia

Gemini 무료 모델
→ provider=gemini

OpenRouter 무료 모델
→ provider=openrouter

DeepSeek 무료/저가 모델
→ provider=deepseek

무료 모델이라고 전부 OpenRouter로 보내면 안 된다.
